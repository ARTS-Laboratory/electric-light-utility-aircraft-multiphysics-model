# Electric Light Utility Aircraft Multiphysics Model

This repository contains a multiphysics model of a light utility electric aircraft.

The model integrates electrical, thermal, and mechanical domains to support powertrain analysis and system-level studies of battery-electric aircraft. It is intended for evaluating energy usage, performance, and design tradeoffs in electric propulsion systems.

This aircraft represents a light utility class (approximately 4–6 seats) with performance characteristics comparable to a Cessna 206. The model is conceptual and is not based on proprietary manufacturer data.

## Scope
- Multiphysics aircraft powertrain model  
- Battery-electric propulsion system  
- Flight energy and load profile simulation  
- Platform for control, thermal, and performance studies  

## Purpose
This model is developed to support research in electric aircraft powertrains, including:
- Energy consumption analysis  
- Thermal behavior  
- System integration and control  
- Design trade studies  

## Notes
This is a research model intended for analysis and development. It may evolve as additional features and validation are added.
