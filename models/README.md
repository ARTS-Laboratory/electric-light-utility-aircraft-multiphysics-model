# Models

## Versions

- **v0.1.0** – Initial development version including multiple multiphysics models (Modelica and Simscape), with early implementations and test cases (including Pipistrel-type configurations).

- **v0.2.0** – Focused Simscape-based multiphysics model of a light utility electric aircraft (206-class), with consolidated aircraft and powertrain representation.

- **v0.2.1** – Cleaned and streamlined version of v0.2.0 prepared for publication, with unnecessary components removed and structure aligned with the 2026 journal paper: *“Battery Degradation and Replacement Cost Impacts on Profitability in Electric Light Utility Aircraft using a Multiphysics Modeling Approach.”*
